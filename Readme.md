# Simple Server

A simple GO server structure that can be used for building production apps

## Dependencies
- GORM
- Gorrilla Mux

## Features
- [X] Routing
- [X] Query builder with the help of GORM
- [X] A modelled MVC structure
- [X] Migration
- [X] Seeders
